Windows 10系统下如何关闭烦人的自动更新

工具 windows 2019/01/27

1、按键盘上的“Windows徽标键+R”组合键，可以调出“运行”窗口

2、在“运行”窗口中输入“gpedit.msc”

3、在“运行”窗口输入“gpedit.mac”后点击“确定”按钮，会弹出“本地策略编辑器(下图)”

4、点击“计算机配置”下的“管理模版”，然后点击“管理模版”下的“Windows 组件”，然后在“Windows 组件”下找到“Windows 更新”，点击“Windows 更新”；然后找到右边列表里面的“配置自动更新（下图）

5、双击运行“配置更新”，会弹出“配置更新”窗口；窗口上面的3个单项按钮，默认选中是“未配置”，这里需要修改为“已禁用（下图）”；然后点击“确定”按钮

6、大功告成
